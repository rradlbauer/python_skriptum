## Übung 11 -- Log-Datei analysieren (mittel)
**Aufgabe:** 
Aus einem fehlerhaften Serverlog hast du als String den Zeitstempel `"2026-08-12 04:15:00"` ausgelesen. Wandle diesen String in ein echtes `datetime`-Objekt um.
**Hilfestellungen:**
* Verwende `datetime.strptime`.

**Musterlösung:**
```python
from datetime import datetime

log_str = "2026-08-12 04:15:00"
log_datum = datetime.strptime(log_str, "%Y-%m-%d %H:%M:%S")
print(type(log_datum), log_datum)
```

## Übung 12 -- Historischer Wochentag (mittel)
**Aufgabe:** 
Die erste Mondlandung fand am 20. Juli 1969 statt. Berechne, welcher Wochentag das war, und gib ihn als Zahl (0 = Montag, 6 = Sonntag) aus.
**Hilfestellungen:**
* Erstelle ein `datetime`-Objekt für diesen Tag und nutze `.weekday()`.

**Musterlösung:**
```python
from datetime import datetime

mondlandung = datetime(1969, 7, 20)
print("Wochentag (0=Montag):", mondlandung.weekday())
```

## Übung 13 -- Garantieablauf (mittel)
**Aufgabe:** 
Ein heute gekauftes Smartphone hat exakt 730 Tage Garantie. Berechne das Ablaufdatum der Garantie und gib es aus.
**Hilfestellungen:**
* Nutze `timedelta(days=730)`.

**Musterlösung:**
```python
from datetime import datetime, timedelta

kaufdatum = datetime.now()
ablaufdatum = kaufdatum + timedelta(days=730)
print("Garantie endet am:", ablaufdatum.strftime("%d.%m.%Y"))
```

## Übung 14 -- Dauer eines Meetings (mittel-schwer)
**Aufgabe:** 
Ein Meeting startete um 09:15 Uhr und endete am selben Tag um 14:45 Uhr (verwende hierfür den heutigen Tag). Berechne die Dauer des Meetings.
**Hilfestellungen:**
* Erzeuge zwei `datetime`-Objekte für heute mit den jeweiligen Uhrzeiten und subtrahiere sie.

**Musterlösung:**
```python
from datetime import datetime

heute = datetime.now()
start = datetime(heute.year, heute.month, heute.day, 9, 15)
ende = datetime(heute.year, heute.month, heute.day, 14, 45)

dauer = ende - start
print("Das Meeting dauerte:", dauer)
```

## Übung 15 -- Altersprüfung für Rabatt (schwer)
**Aufgabe:** 
Ein Kunde ist am `15.05.2007` geboren. Schreibe ein Programm, das das Alter in Tagen berechnet. Wenn die Person älter oder genau 18 Jahre alt ist (zur Vereinfachung: `>= 18 * 365` Tage), gib `"Rabatt gewährt"` aus, sonst `"Kein Rabatt"`.
**Hilfestellungen:**
* Subtrahiere das Geburtsdatum von `datetime.now()` und nutze eine `if`-Abfrage über das `.days` Attribut.

**Musterlösung:**
```python
from datetime import datetime

geburtsdatum = datetime(2007, 5, 15)
jetzt = datetime.now()
alter_in_tagen = (jetzt - geburtsdatum).days

if alter_in_tagen >= (18 * 365):
    print("Rabatt gewährt")
else:
    print("Kein Rabatt")
```

## Übung 16 -- Projekt-Deadline (schwer)
**Aufgabe:** 
Die Deadline für ein großes Projekt ist als String `"2026-12-31 12:00:00"` gegeben. Wandle den String um und berechne, wie viele Tage dem Team ausgehend vom heutigen Zeitpunkt noch bleiben.
**Hilfestellungen:**
* Kombiniere `strptime`, `datetime.now()` und bilde die Differenz.

**Musterlösung:**
```python
from datetime import datetime

deadline_str = "2026-12-31 12:00:00"
deadline = datetime.strptime(deadline_str, "%Y-%m-%d %H:%M:%S")
jetzt = datetime.now()

restzeit = deadline - jetzt
print(f"Es bleiben noch {restzeit.days} Tage bis zur Deadline.")
```

## Übung 17 -- Der nächste Freitag, der 13. (schwer)
**Aufgabe:** 
Ausgehend vom Datum `01.01.2026`, finde den ersten darauf folgenden Freitag, der auf einen 13. Tag im Monat fällt. Gehe das Datum in einer Schleife Tag für Tag durch, bis die Bedingung zutrifft.
**Hilfestellungen:**
* Nutze eine `while`-Schleife, addiere jeweils `timedelta(days=1)`. Prüfe mit `.day == 13` und `.weekday() == 4`.

**Musterlösung:**
```python
from datetime import datetime, timedelta

datum = datetime(2026, 1, 1)
tag_schritt = timedelta(days=1)

while not (datum.day == 13 and datum.weekday() == 4):
    datum += tag_schritt

print("Der nächste Freitag der 13. ist am:", datum.strftime("%d.%m.%Y"))
```
