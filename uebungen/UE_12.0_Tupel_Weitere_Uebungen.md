# Übungen 12.0 – Tupel

Die Aufgaben werden Schritt für Schritt schwieriger.

Versuche jede Aufgabe **zuerst selbst zu lösen**, bevor du dir die Musterlösung ansiehst.

# Übung 1 – Ein Tupel erstellen

Erstelle ein Tupel mit folgenden Werten:

- Name: `"Anna"`
- Alter: `20`
- Größe: `168`

Speichere das Tupel in der Variable `person` und gib es aus.

## Musterlösung

```python
person = ("Anna", 20, 168)
print(person)
```

## Erklärung

- Tupel werden mit **runden Klammern `()`** erstellt.
- Mehrere Werte werden durch **Kommas getrennt**.
- Das Tupel enthält hier verschiedene Datentypen: `str` und `int`.

---

# Übung 2 – Auf Tupel-Elemente zugreifen

Gegeben ist folgendes Tupel:

```python
person = ("Anna", 20, 168)
```

Gib aus:

1. den Namen
2. das Alter
3. die Größe

## Musterlösung

```python
person = ("Anna", 20, 168)

print(person[0])
print(person[1])
print(person[2])
```

## Erklärung

Der **Index** bestimmt die Position:

| Index | Wert |
|------|------|
| 0 | Name |
| 1 | Alter |
| 2 | Größe |

Der erste Index ist immer **0**.

# Übung 3 – Letztes Element

Gegeben ist folgendes Tupel:

```python
zahlen = (4, 7, 2, 9, 5)
```

Gib das **letzte Element** aus, ohne den Index direkt zu zählen.

## Musterlösung

```python
zahlen = (4, 7, 2, 9, 5)

print(zahlen[-1])
```

## Erklärung

Negative Indizes zählen **von hinten**:

| Index | Element |
|------|------|
| -1 | letztes |
| -2 | vorletztes |

# Übung 4 – Tupel entpacken

Gegeben ist folgendes Tupel:

```python
person = ("Max", 19, 180)
```

Entpacke das Tupel in die Variablen:

- `name`
- `alter`
- `groesse`

Gib anschließend alle Variablen aus.

## Musterlösung

```python
person = ("Max", 19, 180)

name, alter, groesse = person

print(name)
print(alter)
print(groesse)
```

## Erklärung

Beim **Entpacken eines Tupels** wird jedes Element einer Variablen zugewiesen.

Die Anzahl der Variablen muss **genau der Anzahl der Tupel-Elemente entsprechen**.

# Übung 5 – Werte ignorieren

Gegeben ist folgendes Tupel:

```python
person = ("Lisa", "12.03.2002", 170, 60)
```

Du möchtest nur:

- den Namen
- die Größe

speichern.

## Musterlösung

```python
person = ("Lisa", "12.03.2002", 170, 60)

name, _, groesse, _ = person

print(name)
print(groesse)
```

## Erklärung

Der Unterstrich `_` wird oft verwendet, wenn ein Wert **nicht benötigt wird**.

Die Position muss trotzdem angegeben werden.

# Übung 6 – Über ein Tupel iterieren

Gegeben ist folgendes Tupel:

```python
farben = ("rot", "grün", "blau", "gelb")
```

Gib alle Farben mit einer `for`-Schleife aus.

## Musterlösung

```python
farben = ("rot", "grün", "blau", "gelb")

for farbe in farben:
    print(farbe)
```

## Erklärung

Die `for`-Schleife durchläuft automatisch alle Elemente des Tupels.

Die Variable `farbe` enthält dabei nacheinander jedes Element.

# Übung 7 – Tupel mit enumerate

Gegeben ist folgendes Tupel:

```python
tiere = ("Hund", "Katze", "Maus")
```

Gib für jedes Tier **Index und Namen** aus.

Beispiel:

```
0 Hund
1 Katze
2 Maus
```

## Musterlösung

```python
tiere = ("Hund", "Katze", "Maus")

for i, tier in enumerate(tiere):
    print(i, tier)
```

## Erklärung

`enumerate()` liefert:

- den **Index**
- das **Element**

Das Tupel wird dabei direkt in zwei Variablen entpackt.

# Übung 8 – Liste von Tupeln

Gegeben ist eine Liste mit Personen:

```python
personen = [
    ("Anna", 21),
    ("Max", 19),
    ("Julia", 23)
]
```

Gib für jede Person folgenden Satz aus:

```
Anna ist 21 Jahre alt
```

## Musterlösung

```python
personen = [
    ("Anna", 21),
    ("Max", 19),
    ("Julia", 23)
]

for name, alter in personen:
    print(name, "ist", alter, "Jahre alt")
```

## Erklärung

Die Liste enthält **Tupel**.

Jedes Tupel wird in der Schleife **direkt entpackt**.

Warum Tupel?

- Jede Person hat eine **feste Struktur**: `(Name, Alter)`

Warum Liste?

- Die Anzahl der Personen kann sich **ändern**.

# Übung 9 – Liste in einem Tupel

Ein Student wird so gespeichert:

```python
student = ("Tom", [2, 1, 3])
```

Die Liste enthält Noten.

Füge die Note `2` hinzu und gib anschließend das Tupel aus.

## Musterlösung

```python
student = ("Tom", [2, 1, 3])

student[1].append(2)

print(student)
```

## Erklärung

Das Tupel selbst ist **unveränderlich**, aber die **Liste im Tupel** kann verändert werden.

Warum?

- Das Tupel beschreibt die **feste Struktur** `(Name, Notenliste)`
- Die Notenliste kann sich **ändern**

# Übung 10 – Neue Tupel erzeugen

Gegeben ist folgendes Tupel:

```python
person = ("Anna", 20, 168)
```

Die Größe soll auf `170` geändert werden.

Da Tupel unveränderlich sind, soll ein **neues Tupel** erstellt werden.

## Musterlösung

```python
person = ("Anna", 20, 168)

name, alter, groesse = person

person_neu = (name, alter, 170)

print(person_neu)
```

## Erklärung

Ein Tupel kann **nicht direkt verändert werden**.

Deshalb:

1. Tupel entpacken
2. gewünschten Wert ändern
3. neues Tupel erstellen

# Zusatzaufgabe (schwer)

Gegeben ist folgende Liste:

```python
punkte = [
    (2, 3),
    (5, 1),
    (4, 7)
]
```

Berechne die **Summe aller x-Werte**.

## Musterlösung

```python
punkte = [
    (2, 3),
    (5, 1),
    (4, 7)
]

summe = 0

for x, y in punkte:
    summe += x

print(summe)
```

## Erklärung

Die Liste enthält **Koordinaten-Tupel** `(x, y)`.

Beim Durchlaufen der Liste werden die Tupel direkt in `x` und `y` entpackt.

So kann man einfach mit den einzelnen Werten arbeiten.
