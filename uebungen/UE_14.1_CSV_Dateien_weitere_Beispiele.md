## Übung 1 -- Personalakte einlesen (einfach)
**Aufgabe:** 
Lies die Datei `personal.csv` ein. Die Werte sind durch einen Strichpunkt (`;`) getrennt. Gib die Tabelle anschließend aus.
**Hilfestellungen:**
* Verwende `read_csv` aus dem Modul `pandas`.

**Musterlösung:**
```python

```

## Übung 7 -- Personalisierte Ausgabe (mittel-schwer)
**Aufgabe:** 
Lies `personal.csv` (Spalten: `Name`, `Abteilung`) ein. Iteriere durch die Zeilen und printe für jeden Mitarbeiter einen Satz im Format: *"Herr/Frau [Name] arbeitet in der Abteilung [Abteilung]."*
**Hilfestellungen:**
* Verwende eine `for`-Schleife und die Methode `iterrows()`.

**Musterlösung:**
```python
from pandas import read_csv

df = read_csv("personal.csv", delimiter=";")

for index, zeile in df.iterrows():
    name = zeile["Name"]
    abteilung = zeile["Abteilung"]
    print(f"Herr/Frau {name} arbeitet in der Abteilung {abteilung}.")
```

## Übung 8 -- Summe des Lagerwerts berechnen (schwer)
**Aufgabe:** 
In `produkte.csv` stehen die Spalten `Preis` und `Bestand`. Iteriere mit einer Schleife durch das DataFrame, multipliziere für jeden Artikel den Preis mit dem Bestand und addiere das auf einen Gesamtlagerwert. Gib den Gesamtwert am Ende aus.
**Hilfestellungen:**
* Lege vor der Schleife eine Variable `gesamtwert = 0` an. Nutze `iterrows()`.

**Musterlösung:**
```python
from pandas import read_csv

df = read_csv("produkte.csv", delimiter=";")
gesamtwert = 0

for index, zeile in df.iterrows():
    wert_pro_artikel = zeile["Preis"] * zeile["Bestand"]
    gesamtwert += wert_pro_artikel

print("Der totale Lagerwert beträgt:", gesamtwert, "Euro.")
```

## Übung 9 -- Produkt mit geringstem Bestand suchen (schwer)
**Aufgabe:** 
Finde mit einer Schleife heraus, welcher Artikel in `produkte.csv` den geringsten `Bestand` aufweist. Gib am Ende den Namen des Artikels und den Bestand aus.
**Hilfestellungen:**
* Definiere vor der Schleife eine Variable `min_bestand` (z.B. mit einem sehr hohen Wert wie 999999 initiiert) und merke dir in einer zweiten Variable den Namen. Vergleiche in der Schleife.

**Musterlösung:**
```python
from pandas import read_csv

df = read_csv("produkte.csv", delimiter=";")
min_bestand = df.Bestand.iloc[0]
artikel_name = ""   # oder None

for index, zeile in df.iterrows():
    aktueller_bestand = zeile["Bestand"]
    if aktueller_bestand < min_bestand:
        min_bestand = aktueller_bestand
        artikel_name = zeile["Artikel"]

print(f"Am wenigsten auf Lager ist '{artikel_name}' mit {min_bestand} Stück.")
```

## Übung 10 -- Filtern nach zwei Bedingungen mit Iteration (schwer)
**Aufgabe:** 
Du möchtest eine E-Mail-Liste aller gut verdienenden Mitarbeiter im Vertrieb erstellen. Gehe das DataFrame `personal.csv` mit einer Schleife durch. Füge den `Name` einer neuen, normalen Python-Liste hinzu, wenn die `Abteilung` gleich `"Vertrieb"` ist **und** das `Gehalt` größer als `3000` ist. Gib die Liste am Ende aus.
**Hilfestellungen:**
* Kombiniere `iterrows()`, ein leeres Listen-Objekt `[]` und eine `if ... and ...`-Abfrage.

**Musterlösung:**
```python
from pandas import read_csv

df = read_csv("personal.csv", delimiter=";")
ziel_personen = []

for index, zeile in df.iterrows():
    if zeile["Abteilung"] == "Vertrieb" and zeile["Gehalt"] > 3000:
        ziel_personen.append(zeile["Name"])

print("Gut verdienende Vertriebler:", ziel_personen)
```
