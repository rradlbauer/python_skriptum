# UE_26.0 Spezielle Methoden - Übungen

### UE_28.0_1 Bruchrechnen

Implementiere eine Klasse `Bruch`, 
welche Brüche mit einem Zähler und einem Nenner repräsentiert.
Die Klasse soll alle speziellen Methoden implementieren,
so dass folgender Code korrekt ausgeführt werden kann:

```python
b1 = Bruch(1, 3)
b2 = Bruch(3, 4)
print(f"{b1} + {b2} = {b1 + b2}")
print(f"{b1} - {b2} = {b1 - b2}")
print(f"{b1} * {b2} = {b1 * b2}")
print(f"{b1} / {b2} = {b1 / b2}")
print(f"{b1} > {b2}: {b1 > b2}")
print(f"{b1} < {b2}: {b1 < b2}")
print(f"{b1} == {b2}: {b1 == b2}")
print(f"{b1} >= {b2}: {b1 >= b2}")
print(f"{b1} <= {b2}: {b1 <= b2}")
```
Außerdem sollen Brüche automatisch gekürzt werden,
und die Ausgabe soll im Format `Zähler/Nenner` erfolgen.

Die Ausgabe sollte dann so aussehen:

```
1/3 + 3/4 = 13/12
1/3 - 3/4 = -5/12
1/3 * 3/4 = 1/4
1/3 / 3/4 = 4/9
1/3 > 3/4: False
1/3 < 3/4: True
1/3 == 3/4: False
1/3 >= 3/4: False
1/3 <= 3/4: True
```
