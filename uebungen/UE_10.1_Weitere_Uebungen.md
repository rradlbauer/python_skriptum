"Berechne den Durchschnitt von drei Zahlen."

2. Die Ausgabe
3. Die notwendigen Schritte in Stichpunkten

## Aufgabe 3 – Algorithmus beschreiben

Gegeben ist folgender Code:

```python
zahl = 1
while zahl <= 5:
    print(zahl)
    zahl = zahl + 1
```

Beschreibe den Algorithmus in Worten.

### Musterlösung

1. Setze die Variable `zahl` auf 1.
2. Wiederhole folgende Schritte, solange `zahl` kleiner oder gleich 5 ist:
   - Gib `zahl` aus.
   - Erhöhe `zahl` um 1.
3. Wenn `zahl` größer als 5 ist, beende den Algorithmus.

## Aufgabe 4 – Vorgegebenen Algorithmus verstehen

Gegeben ist folgender Algorithmus in Worten:

1. Setze eine Variable `summe` auf 0.
2. Gehe alle Zahlen einer Liste durch.
3. Wenn eine Zahl größer als 0 ist, addiere sie zur `summe`.
4. Gib die `summe` aus.

Formuliere diesen Algorithmus in Python.

### Musterlösung

```python
zahlen = [3, -2, 7, -5, 4]
summe = 0

for zahl in zahlen:
    if zahl > 0:
        summe = summe + zahl

print(summe)
```

## Aufgabe 6 – Lückentext (Minimum finden)

Ergänze den Algorithmus, der das kleinste Element einer Liste bestimmt.

```python
zahlen = [8, 3, 6, 2, 9]
minimum = zahlen[0]

for zahl in zahlen:
    if zahl ______ minimum:
        minimum = zahl

print(minimum)
```

### Musterlösung

```python
zahlen = [8, 3, 6, 2, 9]
minimum = zahlen[0]

for zahl in zahlen:
    if zahl < minimum:
        minimum = zahl

print(minimum)
```

## Aufgabe 7 – Anzahl zählen

Schreibe einen Algorithmus, der zählt, wie viele gerade Zahlen in einer Liste enthalten sind.

Formuliere zuerst:
- Eingabe
- Ausgabe
- Schritte

Implementiere danach den Algorithmus in Python.

### Musterlösung

```python
zahlen = [1, 4, 6, 9, 10]
anzahl = 0

for zahl in zahlen:
    if zahl % 2 == 0:
        anzahl = anzahl + 1

print(anzahl)
```

## Aufgabe 8 – Einfache Suche ohne break

Schreibe einen Algorithmus, der überprüft, ob eine Zahl in einer Liste vorkommt.
Verwende keine eingebaute Funktion wie `in`.

### Musterlösung

```python
zahlen = [5, 2, 8, 1]
gesucht = 8
gefunden = False

for zahl in zahlen:
    if zahl == gesucht:
        gefunden = True

if gefunden:
    print("Gefunden")
else:
    print("Nicht gefunden")
```

## Aufgabe 9 – Algorithmus kritisch prüfen

Gegeben ist folgender Algorithmus:

```python
zahlen = [1, 2, 3]
for zahl in zahlen:
    zahl = zahl * 2
```

Frage:
Verändert dieser Algorithmus die Liste? Begründe deine Antwort.

### Musterlösung

Nein, die Liste wird nicht verändert.
Die Variable `zahl` ist nur eine Kopie des jeweiligen Elements.
Um die Liste zu verändern, müsste man über die Indizes gehen und die Elemente direkt ersetzen.

## Aufgabe 10 – Kreativaufgabe: Punktebewertung

Entwickle einen Algorithmus für folgende Situation:

Ein Schüler hat mehrere Testergebnisse (Liste von Punkten).
- Wenn der Durchschnitt mindestens 50 ist, hat er bestanden.
- Sonst nicht.

Formuliere zuerst den Algorithmus in Worten.
Implementiere ihn danach in Python.

### Musterlösung

```python
punkte = [60, 45, 70, 55]
summe = 0

for p in punkte:
    summe = summe + p

durchschnitt = summe / len(punkte)

if durchschnitt >= 50:
    print("Bestanden")
else:
    print("Nicht bestanden")
```
