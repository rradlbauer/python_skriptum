# UE_29.0 Parameter - Übungen

### UE_29.0_1: Rezept-Dosierung

Schreibe eine Funktion `berechne_dosis(gewicht, wirkstoff_konzentration, dosis_pro_kg)`,
die drei Parameter entgegennimmt:
- `gewicht`: Körpergewicht in kg (float)
- `wirkstoff_konzentration`: Konzentration des Wirkstoffs in mg/ml (float)
- `dosis_pro_kg`: Dosis pro kg Körpergewicht in mg/kg (float)

Die Funktion soll die benötigte Dosis in ml berechnen und zurückgeben:
`(gewicht * dosis_pro_kg) / wirkstoff_konzentration`

Teste die Funktion mit verschiedenen Argumenten, sowohl positional als auch mit Schlüsselwortargumenten.

### UE_29.0_2: BMI-Kategorisierung

Schreibe eine Funktion `bmi_kategorie(bmi, untergewicht=18.5, normal=24.9, uebergewicht=29.9)`,
die den BMI-Wert kategorisiert:
- BMI < `untergewicht` → "Untergewicht"
- `untergewicht` <= BMI < `normal` → "Normalgewicht"
- `normal` <= BMI < `uebergewicht` → "Übergewicht"
- BMI >= `uebergewicht` → "Adipositas"

Die Funktion soll die Kategorie als String zurückgeben.

Teste die Funktion:
- Nur mit dem BMI-Wert (Standardwerte verwenden)
- Mit benutzerdefinierten Grenzwerten

### UE_29.0_3: Laborwerte-Statistik

Schreibe eine Funktion `statistik_labor(messung1, *weitere_messungen)`,
welche von einer beliebigen Anzahl an Laborwerten den kleinsten Wert,
den größten Wert und den Mittelwert ermittelt.

Die Funktion soll diese Werte als Dictionary zurückgeben:
```python
statistik_labor(5.2, 4.8, 5.1, 5.3)  
# Rückgabe: {'min' : 4.8, 'max' : 5.3, 'mittel' : 5.1}
```

Teste die Funktion mit 3, 5 und 8 Messwerten.

### UE_29.0_4: Patientenakte

Schreibe eine Funktion `patienten_eintrag(name, geburtsdatum, **daten)`,
die einen Eintrag für eine Patientenakte erstellt:
- Der Eintrag beginnt mit `"Patient: {name}, Geboren: {geburtsdatum}"`
- Danach werden alle zusätzlichen Schlüsselwortargumente
  als weitere Informationen angehängt: `" {key}: {value}"`
- Die Funktion gibt den vollständigen String zurück

Beispiel:
```python
eintrag = patienten_eintrag("Anna Müller", "15.03.1985",
                            groesse=172, gewicht=68, blutgruppe="A+")
# Ergebnis: "Patient: Anna Müller, Geboren: 15.03.1985 groesse: 172 gewicht: 68 blutgruppe: A+"
```

### UE_29.0_5: Medikation berechnen (Kombinationsübung)

Schreibe eine Funktion `medikation_berechnen(*dosierungen, interval=8, einheit="mg")`,
die eine variable Anzahl von Dosierungen (in der angegebenen Einheit) akzeptiert
und folgende Werte als Dictionary zurückgibt:
- `"gesamt_dosis"`: Summe aller Dosierungen
- `"interval"`: Das eingenommene Intervall in Stunden
- `"einheit"`: Die Einheit der Dosierung
- `"tages_dosis"`: Berechnete Tagesdosis: `(24 / interval) * gesamt_dosis`

Beispiel:
```python
medikation_berechnen(500, 500, 500, interval=8, einheit="mg")
# Rückgabe: {"gesamt_dosis": 1500, "interval": 8, "einheit": "mg", "tages_dosis": 4500}
```

Teste die Funktion mit verschiedenen Anzahlen von Dosierungen
und unterschiedlichen Intervallen und Einheiten.
