# Übungsreihe: Datum und Uhrzeit in Python (datetime)

------------------------------------------------------------------------

## Übung 1 -- Aktuelles Datum und Uhrzeit ausgeben (leicht)

**Aufgabe:**\
Schreibe ein Python-Programm, das das aktuelle Datum und die aktuelle
Uhrzeit ausgibt.

**Hilfestellungen:**\
- Verwende `from datetime import datetime`\
- Nutze die Methode `datetime.now()`

**Musterlösung:**

``` python
from datetime import datetime

jetzt = datetime.now()
print(jetzt)
```

------------------------------------------------------------------------

## Übung 2 -- Einzelne Bestandteile ausgeben (leicht)

**Aufgabe:**\
Gib vom aktuellen Zeitpunkt nur das Jahr, den Monat und den Tag aus.

**Hilfestellungen:**\
- Nutze `.year`, `.month`, `.day`

**Musterlösung:**

``` python
from datetime import datetime

jetzt = datetime.now()
print(jetzt.year)
print(jetzt.month)
print(jetzt.day)
```

------------------------------------------------------------------------

## Übung 3 -- Formatierte Ausgabe (leicht-mittel)

**Aufgabe:**\
Gib das aktuelle Datum im Format `TT.MM.JJJJ` aus.

**Hilfestellungen:**\
- Verwende `strftime`\
- Formatcode: `%d.%m.%Y`

**Musterlösung:**

``` python
from datetime import datetime

jetzt = datetime.now()
print(jetzt.strftime("%d.%m.%Y"))
```

------------------------------------------------------------------------

## Übung 4 -- Uhrzeit formatieren (mittel)

**Aufgabe:**\
Gib die aktuelle Uhrzeit im Format `HH:MM:SS` aus.

**Hilfestellungen:**\
- Formatcode: `%H:%M:%S`

**Musterlösung:**

``` python
from datetime import datetime

jetzt = datetime.now()
print(jetzt.strftime("%H:%M:%S"))
```

------------------------------------------------------------------------

## Übung 5 -- Datum aus String erzeugen (mittel)

**Aufgabe:**\
Wandle den String `"2025-12-24 18:30:00"` in ein datetime-Objekt um.

**Hilfestellungen:**\
- Verwende `strptime`\
- Format: `%Y-%m-%d %H:%M:%S`

**Musterlösung:**

``` python
from datetime import datetime

datum_str = "2025-12-24 18:30:00"
datum = datetime.strptime(datum_str, "%Y-%m-%d %H:%M:%S")
print(datum)
```

------------------------------------------------------------------------

## Übung 6 -- Wochentag bestimmen (mittel)

**Aufgabe:**\
Gib den Wochentag als Zahl aus (Montag = 0).

**Hilfestellungen:**\
- Verwende `.weekday()`

**Musterlösung:**

``` python
from datetime import datetime

jetzt = datetime.now()
print(jetzt.weekday())
```

------------------------------------------------------------------------

## Übung 7 -- Alter berechnen (mittel-schwer)

**Aufgabe:**\
Berechne das Alter (in Tagen) für Geburtsdatum 01.01.2000.

**Hilfestellungen:**\
- Subtrahiere datetime-Objekte

**Musterlösung:**

``` python
from datetime import datetime

geburtstag = datetime(2000, 1, 1)
jetzt = datetime.now()
alter = jetzt - geburtstag

print(alter.days)
```

------------------------------------------------------------------------

## Übung 8 -- Datum in der Zukunft berechnen (mittel-schwer)

**Aufgabe:**\
Berechne das Datum in 10 Tagen.

**Hilfestellungen:**\
- `timedelta(days=10)`

**Musterlösung:**

``` python
from datetime import datetime, timedelta

jetzt = datetime.now()
zukunft = jetzt + timedelta(days=10)

print(zukunft)
```

------------------------------------------------------------------------

## Übung 9 -- Differenz zwischen zwei Daten (schwer)

**Aufgabe:**\
Berechne die Tage zwischen 01.01.2020 und 01.01.2025.

**Hilfestellungen:**\
- Subtraktion von datetime

**Musterlösung:**

``` python
from datetime import datetime

d1 = datetime(2020, 1, 1)
d2 = datetime(2025, 1, 1)

print((d2 - d1).days)
```

------------------------------------------------------------------------

## Übung 10 -- Kombination (schwer)

**Aufgabe:**\
Berechne Tage bis zum Termin `"2026-06-01 12:00:00"` und gib ihn
formatiert aus.

**Hilfestellungen:**\
- `strptime`, `strftime`, Differenz

**Musterlösung:**

``` python
from datetime import datetime

termin = datetime.strptime("2026-06-01 12:00:00", "%Y-%m-%d %H:%M:%S")
jetzt = datetime.now()

print("Tage:", (termin - jetzt).days)
print("Termin:", termin.strftime("%d.%m.%Y"))
```
