# 27.0 Grafische Benutzeroberfläche mit Tkinter

## Was ist eine grafische Benutzeroberfläche?

Eine grafische Benutzeroberfläche (GUI) ist eine visuelle Schnittstelle, 
die es Benutzern ermöglicht, mit einem Computerprogramm zu interagieren, 
indem sie grafische Elemente wie Fenster, Schaltflächen, 
Textfelder und Menüs verwenden. 
Im Gegensatz zu einer Konsolen-Anwendung, so wie wir sie bisher
programmiert haben, ermöglicht eine GUI eine intuitivere und 
benutzerfreundlichere Interaktion mit dem Programm.

## Einfaches Beispiel mit Tkinter

Es gibt viele Bibliotheken in Python, um GUIs zu erstellen,
aber eine der beliebtesten und am weitesten verbreiteten ist Tkinter.

Beginnen wir mit einem einfachen Beispiel, um eine GUI mit Tkinter zu erstellen:

```python
import tkinter as tk

def button_click():
    print("Button wurde geklickt!")
    
# Erstelle das Hauptfenster
root = tk.Tk()
root.title("Meine erste GUI")

# Erstelle einen Button und füge ihn zum Fenster hinzu
button = tk.Button(root, text="Klick mich!", command=button_click)
button.pack()

# Starte die Hauptschleife der GUI
root.mainloop()
```
In diesem Beispiel importieren wir das `tkinter`-Modul 
und erstellen eine Funktion `button_click()`,
die ausgeführt wird, wenn der Button geklickt wird.
Wir erstellen das Hauptfenster mit `tk.Tk()` und setzen den Titel.

Dann erstellen wir einen Button mit `tk.Button()`.
Der Konstruktor des Buttons nimmt als Argumente das übergeordnete Fenster (`root`),
den Text, der auf dem Button angezeigt wird, 
und die Funktion, die aufgerufen wird, wenn der Button geklickt wird 
(`command=button_click`).
Wir verwenden die Methode `pack()`, um den Button im Fenster zu platzieren.

Schließlich starten wir die Hauptschleife der GUI mit `root.mainloop()`,
die dafür sorgt, dass das Fenster angezeigt wird 
und auf Benutzerinteraktionen reagiert. Nun wartet das Programm auf Benutzerinteraktionen.

In einer GUI fragt also nicht mehr das Programm den Benutzer nach Eingaben, 
sondern es wartet auf Benutzerinteraktionen, 
wie z.B. das Klicken eines Buttons oder das Eingeben von Text in ein Textfeld,
und reagiert darauf entsprechend.

## Labels zum Anzeigen und Textfelder zum Eingeben von Text

Neben Buttons gibt es viele andere GUI
Elemente, die man verwenden kann, 
z.B. Labels, Textfelder, Checkboxes, Radiobuttons, Menüs, etc. 
Diese Elemente können verwendet werden, 
um komplexere und interaktivere GUIs zu erstellen.

Beispiel für eine GUI, die den Benutzer nach seinem Namen fragt und ihn begrüßt:

```python
import tkinter as tk

def button_click():
    name = entry.get()
    label.config(text=f"Hallo, {name}!")
    
root = tk.Tk()
root.title("Beispiel GUI")
label = tk.Label(root, text="Gib deinen Namen ein:")
label.pack()
entry = tk.Entry(root)
entry.pack()
button = tk.Button(root, text="Greet me!", command=button_click)
button.pack()
root.mainloop()
```

Diese GUI besteht aus einem Label, einem Textfeld (Entry) und einem Button.
Wenn der Button geklickt wird, wird die Funktion `button_click()` aufgerufen,
die den Text aus dem Textfeld mit `entry.get()` liest und das Label aktualisiert,
um den Benutzer mit seinem Namen zu begrüßen.

## Zusammenfassung

### Wichtige Begriffe
- GUI (Graphical User Interface)
- Tkinter
- Widget (Element der GUI, z.B. Button, Label, Entry)
- Eingabe-Elemente (Buttons, Labels, Textfelder, etc.)
- Hauptfenster
- Hauptschleife
- Ereignis (z.B. Button-Klick)
- Callback-Funktion (Funktion, die auf ein Ereignis reagiert)
- Layout (Anordnung der Widgets im Fenster)
- Pack, Grid, Place (verschiedene Layout-Manager)
- Canvas (Zeichenfläche für Grafiken)
- Menü (Dropdown-Menüs, Kontextmenüs)
- Listbox (Anzeige und Auswahl von Listen)



### Das sollst du können
- Ein einfaches GUI mit Tkinter erstellen
- Verschiedene Widgets (Buttons, Labels, Textfelder) verwenden
- Callback-Funktionen definieren, um auf Benutzerinteraktionen zu reagieren
- Die Hauptschleife der GUI starten, um das Fenster anzuzeigen
- Einfache Layouts mit Pack, Grid oder Place erstellen
- Komplexere GUIs mit mehreren Widgets und Interaktionen erstellen
