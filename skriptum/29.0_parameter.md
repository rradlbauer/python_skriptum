# 29.0 Parameter

In Python gibt es viele Möglichkeiten, 
Funktionen und Methoden mit Parametern zu definieren.
Hier sind einige der wichtigsten Möglichkeiten:

### "Normale" Parameter

```python
def addiere(a, b):
    return a + b
```

Das ist die einfachste Form von Parametern, 
bei der beide Parameter `a` und `b` erforderlich sind.
Diese Art von Parametern kennst du bereits. 
Sie werden auch als "positionale Parameter" bezeichnet.
Beim Aufruf der Funktion müssen die 
Argumente in der richtigen Reihenfolge übergeben werden.

### Standardwerte für Parameter

```python
def addiere(a, b=0):
    return a + b
```

In diesem Beispiel hat der Parameter `b` einen Standardwert von `0`.
Das bedeutet, dass `b` optional ist und wenn kein Argument für `b` übergeben wird, 
wird der Standardwert `0` verwendet.
Du kannst die Funktion also auch so aufrufen.

```python
addiere(5)  # Ausgabe: 5, da b den Standardwert 0 verwendet
``` 

Wenn mehrere optionale Parameter vorhanden sind,
sollten sie am Ende der Parameterliste stehen,
da alle erforderlichen Parameter zuerst definiert werden müssen.
Beim Aufruf der Funktion können die optionalen Parameter 
durch Angabe ihres Namens oder durch Position übergeben werden.

Beispiel für die Angabe von optionalen Parametern durch Position:

```python
addiere(5, 3)  # Ausgabe: 8, da b den Wert 3 verwendet
```


### Variable Anzahl von Parametern

```python
def addiere(*args):
    return sum(args)
```
