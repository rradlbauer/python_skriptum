# 29.0 Parameter

Vorab solltest du den Unterschied zwischen Parametern und Argumenten verstehen:
- Parameter: Die Variablen, die in der Funktionsdefinition verwendet werden, um die Eingabewerte zu repräsentieren.
- Argument: Die tatsächlichen Werte, die beim Aufruf der Funktion übergeben werden.

In Python gibt es viele Möglichkeiten, 
Funktionen und Methoden mit Parametern zu definieren.
Hier sind einige der wichtigsten Möglichkeiten:

### "Normale" Parameter

```python
def berechne(a, b, c):
    return (a - b) * c

ergebnis1 = berechne(5, 3, 2)   # Ergebnis: (5 - 3) * 2 = 4
ergebnis2 = berechne(b=3, c=1, a=5)      # Ergebnis: (5 - 3) * 1 = 2
```

Das ist die einfachste Form von Parametern, 
bei der alle Parameter `a` und `b` und `c` erforderlich sind.
Diese Art von Parametern kennst du bereits. 
Sie werden auch als "positionale Parameter" bezeichnet.
Beim Aufruf der Funktion müssen die 
Argumente in der richtigen Reihenfolge übergeben werden.
Alternativ können die Argumente auch durch Angabe ihres Namens übergeben werden,
was die Reihenfolge der Argumente beim Aufruf der Funktion flexibel macht.

### Standardwerte für Parameter

```python
def berechne(a, b = 0, c = 1):
    return (a - b) * c

ergebnis1 = berechne(5)             # Ergebnis: (5 - 0) * 1 = 5
ergebnis2 = berechne(5, 3)          # Ergebnis: (5 - 3) * 1 = 2
ergebnis3 = berechne(5, c = 2)      # Ergebnis: (5 - 0) * 2 = 10
```

In diesem Beispiel haben die Parameter `b` und `c` die Standardwerte `0` bzw. `1`.
Wenn die Funktion `berechne` aufgerufen wird, ohne Werte für `b` und `c` anzugeben, werden die Standardwerte verwendet.
Wenn Werte für `b` und `c` angegeben werden, überschreiben sie die Standardwerte.
Parameter mit Standardwerten werden auch als "optionale Parameter" bezeichnet, 
da sie nicht zwingend erforderlich sind, um die Funktion aufzurufen.

Die Parameter mit Standardwerten müssen am Ende der Parameterliste stehen,
da alle erforderlichen Parameter zuerst definiert werden müssen.
Beim Aufruf der Funktion können die optionalen Parameter
durch Angabe ihres Namens oder durch Position übergeben werden.


### Variable Anzahl von Parametern

```python
def addiere(a, *args):
    return sum(args)

ergebnis1 = addiere(1, 2, 3)        # Ergebnis: 6
ergebnis2 = addiere(1, 2, 3, 4, 5)  # Ergebnis: 15
```

In diesem Beispiel verwendet die Funktion `addiere` den Parameter `a`
und den Parameter `*args`.
Der Parameter `a` ist ein normaler Parameter, der einen Wert erwartet,
während `*args` ein spezieller Parameter ist,
der es ermöglicht, eine variable Anzahl von Argumenten zu übergeben.

Die Argumente werden als Tupel in der Variable `args` gespeichert, 
und die Funktion berechnet die Summe aller Argumente mit der eingebauten 
Funktion `sum()`.
Diese Art von Parametern wird auch als "variadische Parameter" bezeichnet, 
da sie eine variable Anzahl von Argumenten akzeptieren können.

### Schlüsselwortargumente

```python
def beschreibe_person(name, alter, **kwargs):
    beschreibung = f"{name} ist {alter} Jahre alt."
    for key, value in kwargs.items():
        beschreibung += f" {key}: {value}."
    return beschreibung
    
ergebnis1 = beschreibe_person("Alice", 30, beruf="Ingenieurin", hobbys=["Lesen", "Reisen"])
# Ergebnis: "Alice ist 30 Jahre alt. beruf: Ingenieurin. hobbys: ['Lesen', 'Reisen']."
```

In diesem Beispiel verwendet die Funktion `beschreibe_person` die Parameter `name` und `alter` 
sowie den speziellen Parameter `**kwargs`.
Die Parameter `name` und `alter` sind normale Parameter, die Werte erwarten,
während `**kwargs` ein spezieller Parameter ist, der es ermöglicht, 
eine variable Anzahl von Schlüsselwortargumenten zu übergeben.
Die Schlüsselwortargumente werden als `dict` in der Variable `kwargs` gespeichert.

Die Funktion erstellt einen String mit einer Beschreibung der Person, 
indem sie die Schlüssel und Werte des Wörterbuchs durchläuft.
Diese Art von Parametern wird auch als "Schlüsselwortargumente" 
bezeichnet, da sie es ermöglichen, Argumente als Schlüssel-Wert-Paare zu übergeben.

## Zusammenfassung

### Wichtige Begriffe
- Parameter
- Argument
- Positionale Parameter
- Optionale Parameter (Parameter mit Standardwerten)
- Variadische Parameter (Parameter mit `*args`)
- Schlüsselwortargumente (Parameter mit `**kwargs`)

### Das sollst du können
- Funktionen mit verschiedenen Arten von Parametern definieren
- Funktionen mit verschiedenen Arten von Argumenten aufrufen
- Den Unterschied zwischen Parametern und Argumenten verstehen
