# 10.0 Algorithmen

## Definition

[Wikipedia](https://de.wikipedia.org/wiki/Algorithmus):
Ein Algorithmus ist eine eindeutige Handlungsvorschrift 
zur Lösung eines Problems oder einer Klasse von Problemen.
Algorithmen bestehen aus endlich vielen, 
wohldefinierten Einzelschritten.
Damit können sie zur Ausführung in ein Computerprogramm implementiert, 
aber auch in menschlicher Sprache formuliert werden. 
Bei der Problemlösung wird eine bestimmte Eingabe 
in eine bestimmte Ausgabe überführt.

**Einfach erklärt:**  
Ein Algorithmus ist also wie ein Rezept in einem Kochbuch.  
- Es gibt eine klare Reihenfolge von Schritten.  
- Jeder Schritt ist genau definiert und verständlich.  
- Wenn du das Rezept Schritt für Schritt befolgst, erhältst du am Ende das gewünschte Ergebnis.  

Algorithmen können für sehr unterschiedliche Aufgaben genutzt werden, zum Beispiel:  
- Berechnung von Zahlen (z. B. Summe, Durchschnitt)  
- Suchen von Daten (z. B. in Listen oder Datenbanken)  
- Sortieren von Elementen (z. B. alphabetisch oder numerisch)  
- Steuerung von Programmen oder Maschinen (z. B. in Spielen oder Robotern)

## Beispiel

Ein einfaches Beispiel für einen Algorithmus ist die Berechnung der Summe
der Zahlen von 1 bis 100. Wir formulieren zuerst die Lösung in Worten:

1. Ich benötige eine Variable, in welcher ich die Summe speichern kann.  
   Diese hat zu Beginn den Wert 0.
2. Ich wiederhole für alle Zahlen von 1 bis 100 folgenden Schritt:  
   Ich addiere die aktuelle Zahl zur Summe.
3. Anschließend gebe ich die Summe aus.

Diese Lösung lässt sich leicht in Python umsetzen:

```python
sum = 0
for i in range(1, 101):
    sum = sum + i
print(sum)
```

**Erklärung des Codes:**  
- `sum = 0` → Wir legen eine Variable `sum` an und starten bei 0.  
- `for i in range(1, 101):` → Wir durchlaufen alle Zahlen von 1 bis 100.  
  *(Die Funktion `range` erzeugt ein Objekt, das wie eine Liste funktioniert. Der erste Parameter ist die Startzahl, der zweite die Obergrenze, die **nicht mehr enthalten** ist.)*  
- `sum = sum + i` → Wir addieren die aktuelle Zahl `i` zur Summe.  
- `print(sum)` → Wir geben das Ergebnis auf dem Bildschirm aus.  

**Vertiefung:**  
Dieser Algorithmus ist ein einfaches Beispiel für eine **iterative Lösung**, bei der Schritte wiederholt werden. Andere Algorithmen können **rekursiv** sein, also sich selbst aufrufen, oder **entscheidungsbasiert**, wobei Bedingungen den Ablauf steuern.

## Weitere Beispiele

### 1. Minimum einer Liste finden
Algorithmus in Worten:  
1. Ich nehme die erste Zahl als aktuelle „kleinste Zahl“.  
2. Ich gehe alle Zahlen der Liste durch:  
   - Wenn eine Zahl kleiner ist als die aktuelle kleinste Zahl, aktualisiere sie.  
3. Am Ende habe ich die kleinste Zahl gefunden.

Python-Code:

```python
zahlen = [5, 8, 2, 9, 1, 7]
min_zahl = zahlen[0]

for zahl in zahlen:
    if zahl < min_zahl:
        min_zahl = zahl

print("Die kleinste Zahl ist:", min_zahl)
```

### 2. Einfache Suche
Algorithmus in Worten:  
1. Ich habe eine Liste von Zahlen und eine Zahl, die ich suchen möchte.  
2. Ich prüfe jede Zahl der Liste:  
   - Wenn die Zahl gefunden wird, stoppe die Suche.  
3. Ergebnis ausgeben.

Python-Code:

```python
zahlen = [4, 7, 1, 9, 3]
gesucht = 9
gefunden = False

for zahl in zahlen:
    if zahl == gesucht:
        gefunden = True
        break

if gefunden:
    print(gesucht, "wurde gefunden.")
else:
    print(gesucht, "wurde nicht gefunden.")
```

## Zusammenfassung

### Wichtige Begriffe
- **Algorithmus** → Handlungsvorschrift zur Lösung eines Problems  
- **Problem** → Die Aufgabe, die gelöst werden soll  
- **Lösung** → Das Ergebnis oder die Vorgehensweise, die das Problem löst  
- **Schritt** → Einzelne Anweisung innerhalb des Algorithmus  

### Das sollst du können
- Für ein Problem einen Algorithmus in Worten formulieren  
- Einen Algorithmus in Python umsetzen  
- Iterative Prozesse mit `for` und `while` verstehen  
- Einfache Entscheidungsstrukturen mit `if` anwenden
